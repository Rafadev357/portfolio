TODOSPORT - E-commerce de Artículos Deportivos

TODOSPORT es una aplicación web de una sola página (SPA) que simula la interfaz de una tienda online, construida íntegramente con React. El proyecto demuestra el manejo de un catálogo de productos y un carrito de compras funcional, utilizando únicamente las herramientas nativas de la librería. Para la gestión del estado global, como el contenido del carrito, implementé el Context API de React (useContext), creando proveedores y contextos personalizados para distribuir los datos de forma eficiente a través de los componentes. Los datos de los productos se manejan localmente, y la navegación entre las diferentes secciones de la web se gestiona con React Router.
